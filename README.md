# python-edu-coin-trading

간단한 암호화폐 트레이딩 교육용 파이썬 프로젝트입니다.

## 소개

이 저장소는 파이썬 기본 문법부터 데이터 시각화, 백테스팅까지 단계별로 학습할 수 있도록 구성된 교육용 자료입니다. 주요 내용은 `doc/` 폴더의 한글 챕터 문서와 `src/` 폴더의 예제 코드 및 노트북으로 구성됩니다.

## 폴더 구조

- `doc/`
  - 파이썬 학습용 한글 챕터 문서
- `src/`
  - 파이썬 예제 스크립트 및 Jupyter Notebook

## 빠른 시작

1. Python 3을 설치합니다.
2. 필요한 라이브러리를 설치합니다.

```bash
pip install pandas matplotlib jupyter requests
```

3. 예제 스크립트를 실행합니다.

```bash
python3 src/chapter_02.py
python3 src/chapter_03.py
```

4. Jupyter Notebook을 열어 노트북 예제를 확인합니다.

```bash
cd src
jupyter notebook
```

## 주요 내용

- `doc/Chapter 02. 파이썬 기초 문법 (Python 3).md`
- `doc/Chapter 03. 코드의 재사용: 함수, 객체, 라이브러리.md`
- `doc/Chapter 04. 데이터 시각화: 내 코인 차트 그리기.md`
- `doc/Chapter 05. 백테스팅: 과거 데이터로 내 전략 검증하기.md`
- `src/chapter_04.ipynb`
- `src/chapter_05.ipynb`
