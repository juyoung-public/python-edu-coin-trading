# Chapter 02. 파이썬 기초 문법 (Python 3)

본격적으로 자동매매 프로그램을 만들기 위해 꼭 필요한 파이썬의 핵심 문법을 배웁니다. 고등학생 수준에서 코인 데이터를 다룰 때 가장 많이 사용하는 기능 위주로 구성했습니다.

---

## 1. 변수와 자료형: 데이터를 담는 상자 📦

컴퓨터 메모리에 데이터를 저장하고 이름을 붙여주는 것을 **변수**라고 합니다.

```python
# 1. 문자열 (String): 글자 데이터
coin_name = "Bitcoin"

# 2. 정수 (Integer): 소수점이 없는 숫자
current_price = 95000000

# 3. 실수 (Float): 소수점이 있는 숫자
change_rate = 2.45

# 4. 불리언 (Boolean): 참(True) 또는 거짓(False)
is_rising = True
```

---

## 2. f-string: 깔끔하게 출력하기 📝

Python 3.6부터 도입된 **f-string**은 문자열 사이에 변수를 넣는 가장 세련된 방법입니다.

```python
name = "비트코인"
price = 95000000

# 문자열 앞에 f를 붙이고 변수를 { }로 감쌉니다.
# {price:,} 처럼 쓰면 천 단위마다 콤마(,)를 자동으로 찍어줍니다.
print(f"{name}의 현재 가격은 {price:,}원입니다.")
```

---

## 3. 리스트와 딕셔너리: 데이터 묶어서 관리하기 📚

자동매매봇은 여러 코인의 이름이나 가격 정보를 한꺼번에 다뤄야 합니다.

### (1) 리스트 (List)
순서가 있는 데이터의 목록입니다.
```python
tickers = ["BTC", "ETH", "XRP", "DOGE"]

print(tickers[0])  # 첫 번째 항목: BTC (0부터 시작함에 주의!)
print(tickers[-1]) # 마지막 항목: DOGE
```

### (2) 딕셔너리 (Dictionary)
'키(Key)'와 '값(Value)'이 한 쌍으로 이루어진 구조입니다. 사전에서 단어(Key)와 뜻(Value)을 찾는 것과 같습니다.
```python
coin_info = {
    "name": "Ethereum",
    "ticker": "ETH",
    "opening_price": 4500000
}

print(f"코인 이름: {coin_info['name']}")
```

---

## 4. 조건문 (If): 판단하기 🚦

특정 조건이 만족될 때만 코드를 실행합니다. 매수/매도 타이밍을 결정할 때 필수입니다.

```python
target_price = 100000000
current_price = 105000000

if current_price >= target_price:
    print("🚀 목표가 돌파! 매수 주문을 전송합니다.")
elif current_price > 90000000:
    print("👀 가격이 오르고 있습니다. 감시를 계속합니다.")
else:
    print("💤 아직 때가 아닙니다.")
```

---

## 5. 반복문 (For): 노가다 탈출하기 🔁

리스트에 들어있는 수많은 데이터를 하나하나 자동으로 처리합니다.

```python
tickers = ["BTC", "ETH", "XRP"]

# tickers 리스트 안의 종목을 하나씩 t라는 변수에 넣으며 반복
for t in tickers:
    print(f"현재 {t} 종목의 시세를 빗썸 API에서 가져오는 중...")

print("조회 완료!")
```

---

## 💡 분석 포인트 (고등학생을 위한 팁)

1.  **파이썬은 들여쓰기(Indentation)가 생명입니다.** `if`나 `for` 다음 줄은 반드시 `Tab`키나 `Space` 4칸으로 들여써야 합니다. 안 그러면 에러가 나요!
2.  **주석(#)을 활용하세요.** 코드 옆에 `#`을 쓰고 설명을 적어두면 나중에 코드를 다시 볼 때 이해하기 훨씬 쉽습니다.
