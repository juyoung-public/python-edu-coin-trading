# Chapter 01. 프로그래밍 입문 및 개발 환경 구축

이 장에서는 프로그래밍의 기초 개념을 이해하고, 나만의 코인 매매 봇을 만들기 위한 파이썬(Python) 개발 환경을 Windows 컴퓨터에 설정합니다.

---

## 1. 프로그래밍이란 무엇인가? 🤖

**프로그래밍**은 쉽게 말해 **'컴퓨터에게 일을 시키기 위해 설계도를 그리는 작업'**입니다.

*   **컴퓨터는 매우 똑똑하지만 바보입니다:** 컴퓨터는 복잡한 계산을 순식간에 해내지만, "대충 알아서 코인 좀 사줘"라는 말은 알아듣지 못합니다. 
*   **구체적인 레시피:** 라면을 끓일 때 '물 올리기 -> 불 켜기 -> 스프 넣기' 순서가 있듯, 컴퓨터에게도 아주 상세하고 논리적인 단계별 지시를 내려야 합니다. 
*   **자동화의 힘:** 사람이 24시간 내내 차트를 볼 수는 없지만, 프로그래밍을 이용하면 컴퓨터가 잠도 자지 않고 우리 대신 시장을 감시하게 할 수 있습니다.

---

## 2. 왜 파이썬(Python 3)인가? 🐍

우리는 수많은 프로그래밍 언어 중 왜 **파이썬**을 배울까요?

1.  **배우기 쉽다:** 파이썬 문법은 영어 문장과 비슷해서 초보자가 읽고 쓰기에 매우 직관적입니다.
2.  **데이터와 금융의 언어:** 전 세계 데이터 과학자와 금융 트레이더들이 가장 많이 사용하는 언어입니다. 우리가 사용할 빗썸 API나 수익률 분석 도구들이 파이썬으로 아주 잘 만들어져 있습니다.
3.  **강력한 커뮤니티:** 모르는 것이 생겼을 때 구글이나 ChatGPT에 물어보면 가장 많은 해답을 찾을 수 있는 언어입니다.

---

## 3. Windows에 파이썬 설치하기 🪟

컴퓨터가 파이썬 언어를 해석할 수 있도록 '해석기(Interpreter)'를 설치해야 합니다.

1.  **공식 사이트 접속:** [python.org](https://www.python.org/)의 Downloads 메뉴에서 Windows용 최신 버전을 다운로드합니다.
2.  **설치 파일 실행:** 다운로드한 `.exe` 파일을 실행합니다.
3.  **⭐⭐가장 중요한 단계⭐⭐:** 설치창 하단의 **`Add Python to PATH`** 체크박스를 **반드시 체크**하세요! (이걸 안 하면 나중에 컴퓨터가 파이썬을 찾지 못합니다.)
4.  **Install Now 클릭:** 설치가 완료될 때까지 기다립니다.

---

## 4. 코드 편집기: Visual Studio Code (VS Code) 💻

요리사에게 잘 드는 칼이 필요하듯, 개발자에게는 코드를 깔끔하게 작성할 수 있는 **에디터**가 필요합니다.

1.  **설치:** [code.visualstudio.com](https://code.visualstudio.com/)에서 Windows용 설치 파일을 받아 설치합니다.
2.  **한국어 설정 (선택):** 왼쪽 아이콘 메뉴(Extensions)에서 `Korean`을 검색하여 'Korean Language Pack'을 설치하면 메뉴가 한글로 바뀝니다.
3.  **파이썬 확장 도구 설치 (필수):** 
    *   왼쪽 사이드바의 테트리스 블록 모양 아이콘(**Extensions**)을 클릭합니다.
    *   검색창에 `Python`을 입력합니다.
    *   Microsoft에서 만든 **Python** 확장 프로그램을 찾아 **[Install]** 버튼을 누릅니다.

---

## 5. 터미널(Terminal) 사용해보기 ⌨️

터미널은 마우스 클릭 대신 '글자'로 컴퓨터에게 명령을 내리는 창입니다.

*   **실행법:** VS Code 상단 메뉴에서 `터미널(Terminal)` -> `새 터미널(New Terminal)`을 클릭합니다.
*   **확인:** 아래쪽 검은 창에 다음 명령어를 입력하고 엔터를 눌러보세요.
    ```bash
    python --version
    ```
*   `Python 3.x.x`라고 버전 정보가 출력된다면 설치가 완벽하게 된 것입니다!

---

## 6. 첫 번째 파이썬 코드 실행 🚀

이제 첫 번째 코드를 작성하고 실행해봅시다.

1.  VS Code에서 `Ctrl + N`을 눌러 새 파일을 만듭니다.
2.  `Ctrl + S`를 눌러 파일 이름을 `hello.py`로 저장합니다. (확장자 `.py`가 중요합니다!)
3.  아래 코드를 입력하세요.
    ```python
    # 내 첫 번째 자동매매 준비 코드
    print("안녕하세요! 코인 자동매매 봇 만들기 프로젝트를 시작합니다.")
    
    a = 1000000  # 내 원금 100만원
    profit_rate = 1.1  # 10% 수익 시
    print(f"10% 수익 시 예상 자산: {a * profit_rate}원")
    ```
4.  **실행:** 오른쪽 상단의 **▷ (Run Python File)** 버튼을 누르거나, 터미널에 `python hello.py`라고 입력합니다.
